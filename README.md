# Project-4----FEMA-NRI-Potential-Bias-Analysis
The repository contains the needed files and information for the CIVE 202 project #4 - NRI Potential Bias Analysis. This code aims to see if the NRI methodology is consistent with an alternative risk model that evaluates to geographically adjacent states, Arkansas and Missouri. 

The goal of this project is to quantitatively analyze the bias within the NRI statistics. We will compare it against an alternative risk measurement system to find any inconsistencies between the two datasets. The demographic and spatial profiles for these states are different but we will be running the FEMA NRI and our alternative to compare the results. From this data we will have our conlcusion on whether there is inherent bias in the FEMA NRI results against our alternative. WE have put these data points into tables and grpahics that make the visual analysis easy to do. 

To use this python tool you will need to download the files in this repository and follow the directions in the documents provided.
